Project Gallery

chainladder-python

chainladder-python is a Python package designed to handle all core actuarial reserving tasks, including triangle data manipulation, link ratio calculations, and IBNR estimation using both deterministic and stochastic models. It offers a modern alternative to outdated legacy tools, with a syntax inspired by familiar libraries like pandas and scikit-learn. Built to be intuitive, minimal, and powerful, it enables actuaries to focus on analysis—not software limitations.

Colonizer

Colonizer is a strategic analysis tool for the board game Settlers of Catan. It helps players, especially beginners, make better decisions by identifying suboptimal settlement placements and trades using economic theory. By analyzing game data and common patterns, it offers insights into gameplay mistakes, helping users improve strategy, resource efficiency, and overall competitiveness.

Aerial Extractor

Aerial Extractor is a utility that downloads Apple’s cinematic aerial screensavers, originally created for Apple TV. It allows users to view these high-quality visuals on their own systems by extracting and organizing the entire collection. Lightweight and easy to use, it brings Apple’s most beautiful motion backdrops to desktops and custom displays with minimal setup.

Byte-It

Byte-It is an arcade-style survival game where players dodge moving obstacles while collecting apples and gems for points. Built on custom hardware with microcontrollers, LCD screens, and NES-style controls, it challenges players to stay alive and use power-ups strategically.
